- get normalise css file from its webpage for some standard styles.
- get the font family 'work sans' from google fonts, its link tags and styles to apply.
- install font awesome web icons. copy the 'webfonts' folder to assets and 'all.min.css' to CSS project folder.
- 'all.min.css' is the minimized file of 'all.css' which embedded to use icons in webfonts.
- there are some built-in size classes for i tag : fa-xs (extra-small), fa-sm (small), fa-lg (large), fa-2x to fa-10x (multiple sizes).
- the default value for 'z-index' is auto .works with the positioned elements (has position value other than static). to move element forward give value '1' or '-1' to move backward. 
- to center text elements (headings, paragraphs or images) in a container use 'text-align', but to center other containers use flex-box or position.
- flex-basis differ from actual 'width' in that it is a flexbox-only property. Defines the initial main size of a flex item before free space is distributed.
